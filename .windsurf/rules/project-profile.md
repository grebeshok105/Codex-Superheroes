## Цель мода
- Короткое описание: мод на тематику супергероев с упором на высококачественные VFX, кастомный HUD и проработанную физику движений. Система трансформации завязана на уникальных предметах, меняющих модель и характеристики игрока.
- Жанр/категория: супергерои / способности / VFX
- Главные фичи:
  - Поэтапная реализация 5 героев: Homelander, Iron Man, Batman, Captain America, Doomsday.
  - Первая реализация начинается с Homelander.
  - Кастомные модели накладываются поверх скина как броня и меняют хитбоксы.
  - Динамическое изменение размера (scale) для определённых персонажей, например Doomsday.
  - Инерционный полёт с физикой разгона.
  - Двойная система ресурсов: энергия с авторегенерацией и мана с восстановлением через предметы.
  - Кастомизация управления: игрок сам назначает, какой ресурс тратит конкретная способность.
  - Круговое меню выбора способностей в реальном времени.

## Контент
- Предметы трансформации: уникальный артефакт для каждого героя, активирующий смену формы.
- Расходники маны: специфические предметы для каждого героя для восполнения шкалы маны.
- Homelander: полёт, лазеры из глаз, рентгеновское зрение через эффект Glowing.

## Механики
- Трансформация: при использовании предмета игроку назначается модель героя и соответствующий хитбокс.
- Лазеры Homelander: визуальный летящий снаряд с красивым трасером; наносит урон, но не разрушает блоки.
- Рентген: активация ванильного эффекта подсветки сущностей сквозь препятствия.
- Ресурс-менеджмент: если лимит выбранного ресурса (энергии) исчерпан, система автоматически переключается на потребление маны.
- Пассивные навыки: каждый герой обладает набором постоянных бонусов (сопротивление урону, сила, иммунитет к падению) без затрат ресурсов.

## Чего быть НЕ должно
- Разрушение блоков лазерами.
- Урон от столкновений с блоками при инерционном полёте.
- Пауза времени при открытии меню способностей.

## Нестандартные решения проекта
- Кастомные модели накладываются поверх скина как броня, но при этом меняют хитбокс игрока.
- Двойной ресурс способностей с авто-переключением энергия → мана.
- Игрок сам выбирает, какой ресурс тратит конкретная способность.
- Круговое меню работает в реальном времени без остановки игры.
- Для Doomsday планируется динамическое изменение scale.

## Открытые места
- Конкретный перечень пассивных способностей для каждого из 5 героев.
- Визуальный стиль и ассеты будут предоставлены пользователем в процессе.
- Специфические механики для Batman, Iron Man, Captain America и Doomsday, кроме Homelander.
