# План v3.20.0 — иконки способностей, радиалка v2, полировка панелей

База: ветка `viktor/hud-v3-reinhard-ram` (PR #13). Версия: 3.19.0 → **3.20.0**. Выкат: новый PR.

## 1. Иконки способностей для ВСЕХ героев
- Сейчас текстуры-иконки есть только у 6 способностей (Homelander); остальные — цветные кружки.
- Рисую 64x64 пиксель-арт иконку КАЖДОЙ способности всех 20 героев (~140 шт):
  генератор на Python (общая стилистика: тёмная плитка + скруглённая рамка в цвете темы героя,
  как у бейджей A/T, + уникальная пиктограмма по смыслу способности — меч, кулак, щит, молния,
  глаз, крыло, часы, череп, огонь, лёд и т.п.), затем ручная доводка спорных.
- Куда ставлю (как обведено):
  - **панель подсказок (H)** — иконка способности вместо буквы в рамке; маркер A/T остаётся
    мелким уголком на плитке;
  - **панель героя** — иконка перед именем способности (вместо зелёных точек);
  - **радиальное меню** — иконки в секторах.

## 2. Радиальное меню — полностью с нуля (v2)
- Новый рендер вместо текущего «пирога»:
  - разделённые ДУГОВЫЕ сегменты с зазорами (не сплошной диск), тонкая неоновая кромка;
  - плавное открытие: scale+fade с ease-out, секторы появляются по кругу за ~150мс;
  - hover: сектор плавно «вырастает» наружу и подсвечивается glow, иконка увеличивается;
  - выбранный сектор тянет тонкий луч к курсору; курсор-точка остаётся;
  - имя наведённой способности + КД — НИЖЕ кольца на отдельной плашке (не в хабе);
  - хаб ПУСТОЙ и чистый: никакого текста (имя героя и «Смотри по сторонам…» удаляю),
    только лёгкое стекло + курсор; возврат в центр = отмена (как сейчас).
- Вся логика выбора/каста/блокировки слотов сохраняется, меняется только вид и анимации.

## 3. Энергия — тонкая полоска как HP
- В панели героя сегментированный бар (5px, 10 сегментов) заменяю той же тонкой
  полоской 3px, как у HP. Иконка энергии, название и % остаются.

## 4. Меч Рейнхарда в ножнах — перевернуть
- Лезвие должно смотреть в сторону живота (вперёд-вверх), не вниз: поправить
  rotZ у предмета в `ReinhardScabbardLayer` (~180° к текущему).

## 5. Панель подсказок (H) — лёгкий рефреш
- Плашка-заголовок с именем героя и тонкой акцентной линией снизу;
- вертикальная акцентная полоса слева у секций («Пассивные», «Активные», «Переключаемые»);
- иконки из п.1 вместо буквенных бейджей; чуть больше воздуха между плитками;
- ВСЕ фишки сохраняются: глитч Регулуса (скрытая способность), статусы ГОТОВ/ВЫКЛ, КД.

## 6. 3D-герой в панели — реальная поза
- Сейчас бюст следит за курсором мыши (`renderEntityInInventoryFollowsMouse`).
- Меняю на рендер с РЕАЛЬНОЙ позой игрока: фактические yaw/pitch тела и головы,
  анимации (присед, замах, полёт, поза трансформации) — «зеркало» героя в пространстве,
  повёрнутое лицом к зрителю с фиксированного ракурса.

## 7. Сборка и выкат
- `mod_version` → 3.20.0, сборка, живой чеклист, новый PR (стек на #13),
  jar в тред + GitHub Release v3.20.0 (по постоянному правилу).
