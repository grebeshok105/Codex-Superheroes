# Pandora v2 — кинематик через шейдер, POV-swap, иконки, и пр. (батч из 11 пунктов)

Ветка: `viktor/pandora-house-of-vanity` (продолжение PR #32). Один jar + один PR + релиз.
Тестировать in-game не могу (нет рантайма MC) — числа/визуал тюнит пользователь.

## Задачи
1. **Иконки способностей Пандоры** — сгенерировать PNG (16x16/32x) для: дом тщеславия, каприз творца, пространственная привязка, сжатие пространства, лишение тщеславием. Положить в `assets/superheroes/textures/gui/abilities/`. Убрать буквенные плейсхолдеры (T/A) для Пандоры.
2. **Скрыть все способности Пандоры, открывать только в Доме.** «Дом тщеславия» всегда видим (вход). Остальные 4 — только когда у игрока активен House (`hasActiveHouse`). Гейт в HUD-панели способностей (видимость) + по возможности и в роутере (нельзя активировать вне Дома — для боевых уже есть).
3. **Preset 9 (MODE 9) scale=4** — `MirrorDimensionController.SCALE_CYCLE` последний элемент 256→4.
4. **Вернуть Кадзуху и Скарамуччу** — найти, где они исключены (меню выбора героя/книга/список), вернуть.
5. **Сайт: все персонажи** (+ Пандора, Кадзуха, Скарамучча). Найти исходник сайта (репо grebeshok.eu.cc).
6. **Шейдер кинематика** (Veil post): блоки и сущности → абсолютно чёрные силуэты (без деталей), фон (небо/пустота) → красно-белый градиент как на референсе. Включается только во время кинематики у обоих игроков.
7. **Пандора не умирает.** Перехват урона: HP не опускается ниже 0.5 → стартует кинематик. Во время кинематики ВСЕ игроки замирают (не могут двигать камерой). 
8. **POV-swap + F1.** Игрок-Пандора `setCamera(killer)` — видит глазами убийцы. У обоих принудительно F1 (скрыть ВСЕ элементы HUD без исключений) на время кинематики.
9. **Подъём + чёрные молнии.** Пандора поднимается на 2 блока, вокруг бьют молнии без звука, рендерятся чёрным силуэтом (через тот же шейдер — это сущности молнии / партиклы).
10. **Финал через 5с.** Резкий конец: Пандора телепортируется за спину убийцы, хихиканье, кинематик выключается. Всю кинематику — полный лок ввода (мышь+клава) у обоих, камера Пандоры = глаза убийцы.
11. **Статы Пандоры = голый Стив.** 20 HP, 0 брони/тафнес/бонусов. Правка в PandoraHero attributes.

## Решения/допущения
- #8 POV: спектатор-камера `ServerPlayer.setCamera(killerEntity)` (нельзя 1:1 чужой экран).
- #6: Veil post-pass, не Iris-шейдерпак (чтобы работало всегда).
- #2: гейт только для Пандоры; «Дом тщеславия» всегда виден.
- Кинематик длится ~5с (#10). Старая «вечная неуязвимость до перепoutа героя» — заменяется на «не умирает никогда» (#7), т.е. после кинематики Пандора жива (HP восстановлено).

## Заметки реализации
- Перехват: уже есть `PandoraDeathController` на `ALLOW_DEATH`. Меняем подход на `ServerLivingEntityEvents.ALLOW_DAMAGE` (или ALLOW_DEATH) — гасим урон до 0.5, ловим момент.
- Freeze всех + input lock: клиентские миксины на KeyboardHandler/MouseHandler + блок Input во время кинематики; сервер шлёт payload «кинематик ON/OFF» всем участникам/наблюдателям.
- F1: установить `Minecraft.options.hideGui=true` на клиенте на время.
